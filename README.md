# HP Superbot

A production-ready Discord.js v14 multipurpose bot with moderation, economy, gambling, shop, auto-mod, music, help, joke, and AI features.

## Features

- **Moderation**: Ban, kick, timeout, clear, lock, unlock, warn, warnings, mute, unmute
- **Info**: Ping, userinfo, serverinfo, avatar
- **Economy**: Balance, daily, pay, givebucks, shop, buy, inventory
- **Gambling**: Coinflip, slots, dice, roulette
- **Leaderboards**: Leaderboard, leaderboardbucks
- **Engagement**: 8ball, poll, qotd, remind, joke, ai
- **Music**: Play, leave
- **Config**: Setconfig, viewconfig
- **Giveaway**: Giveaway
- **Help**: Help

## Setup

1. Clone the repository
2. Run `npm install`
3. Fill in `.env` with your bot token, client ID, guild ID, and log channel ID
4. Run `npm run deploy` to register slash commands
5. Run `npm start` to start the bot

## Deployment

This bot is designed to work on Railway or similar platforms. Ensure your environment variables are set.
